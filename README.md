# 📚 PDF Query Bot - Ask Questions About a Research Paper

[![Open in Hugging Face](https://img.shields.io/badge/🤗-Try%20the%20Bot%20Live-yellow)](https://huggingface.co/spaces/amanfisseha/PDF-Bot)

A smart chatbot that answers questions about a **215-page undergraduate research paper** on building a digital AI data marketplace and crowdsourcing platform (called "FidelAI").

Instead of reading the entire document, just ask the bot anything and it will find the answer for you.

## 🎯 What This Bot Does

This bot uses **Retrieval-Augmented Generation (RAG)** to:
1. Read your question
2. Search the research paper for relevant sections
3. Generate an answer based ONLY on what the paper says

## 💬 Example Questions You Can Ask

Try asking the bot any of these:

| Category | Example Question |
|----------|------------------|
| **Architecture** | "What software architecture does the platform use?" |
| **Features** | "What are the main features of the FidelAI platform?" |
| **User Roles** | "Who are the different users of this system?" |
| **Quality Control** | "How does the platform ensure data quality?" |
| **Technology Stack** | "What technologies were used to build this platform?" |
| **Motivation** | "Why was this platform created?" |
| **Contributors** | "Who contributes data to the marketplace?" |
| **Validation** | "How are annotations validated?" |
